# Advanced Student Result Management System

## 📌 Project Overview
The Advanced Student Result Management System is a Core Java based console application developed to manage student academic records efficiently. It helps administrators add, search, update, delete, and maintain student result details with automatic calculations.

## 🚀 Features
- Admin login system
- Add student records
- Search student by roll number
- Update student marks
- Delete student details
- Calculate total marks
- Calculate percentage
- Generate grade automatically
- Show topper details
- Save records using file handling
- Load records during next execution

## 🛠️ Technologies Used
- Core Java
- OOP Concepts
- Collections Framework (ArrayList)
- File Handling
- Exception Handling
- Eclipse IDE

## ▶️ How to Run
1. Open project in Eclipse IDE
2. Run the Main class
3. Login as Admin
4. Use menu options to manage student records
